# Sine Wave Audio Generator

**Онлайн-генератор синусоидального звукового сигнала с возможностью выбора частоты от 20 Гц до 20 кГц и сохранением результата в WAV-файл с настраиваемой длительностью.**

Этот инструмент позволяет создавать чистый синусоидальный звуковой сигнал заданной частоты. Пользователь может плавно изменять частоту с помощью ползунка или задать точное значение вручную, выбрать продолжительность сигнала и скачать готовый WAV-файл.

### Для чего нужен генератор?

Синусоидальный сигнал является базовым элементом в акустике, электронике и обработке звука. Такой генератор позволяет быстро создавать тестовые тоны для проверки аудиооборудования, настройки акустических систем, изучения характеристик звука и проведения экспериментов без использования специализированного программного обеспечения.

### Как пользоваться

1. Откройте приложение:  
   https://volkovhl.github.io/sine-wave-generator/

2. Выберите необходимую частоту сигнала:
   - Используйте ползунок для плавной настройки частоты
   - Или введите точное значение вручную

3. Установите частоту в диапазоне:
   - Минимум: **20 Гц**
   - Максимум: **20 000 Гц (20 кГц)**

4. Задайте длительность создаваемого аудиофайла:
   - Минимальная длительность: **1 секунда**
   - Максимальная длительность: **1000 секунд**

5. Нажмите кнопку генерации.

6. Скачайте готовый аудиофайл в формате **WAV** с выбранной частотой и длительностью.

### Формула генерации сигнала

Синусоидальный звуковой сигнал описывается формулой:

**y(t) = A × sin(2πft)**

где:

- **A** — амплитуда сигнала;
- **f** — частота сигнала (Гц);
- **t** — время;
- **2πf** — угловая частота.
